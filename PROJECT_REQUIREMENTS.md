### 4. 计算逻辑规则
```typescript
// 余额积分规则
$100 - $1000    → 1分
$1000 - $10000  → 2分  
$10000 - $100000 → 3分
$100000+        → 4分

// 交易积分规则 (对数计算)
积分 = floor(log₂(交易金额))
$2 → 1分, $4 → 2分, $8 → 3分, $16 → 4分...
最高15分 ($32768)

// 交易计算
每次交易金额 = 用户余额 × 2
所需交易次数 = ceil(目标交易金额 ÷ 单次交易金额)
建议交易次数 = 所需次数 + 1 (防止磨损卡线)
```
